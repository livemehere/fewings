# useInterval

A hook that sets up an interval to call a callback function at a specified delay. The interval is automatically cleaned up when the component unmounts.

> `callback` uses [`useCallbackRef`](/docs/react/hooks/useCallbackRef) to create a stable reference for the callback function.

## API

```tsx
declare const useInterval: (callback: () => void, delay: number) => void;
```

## Example

```tsx
import { useInterval } from '@fewings/react/hooks';
import { useState } from 'react';

function Timer() {
  const [count, setCount] = useState(0);

  useInterval(() => {
    setCount((prevCount) => prevCount + 1);
  }, 1000); // Call every 1 second

  return <div>Count: {count}</div>;
}
```
