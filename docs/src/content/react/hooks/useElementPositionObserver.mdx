# useElementPositionObserver

A hook that observes element position changes using `requestAnimationFrame`.

> ⚠️ **Warning**: This hook continuously calls `requestAnimationFrame` while active,
> which can be performance intensive. Use with caution and only when necessary.
> Consider using `ResizeObserver` or `IntersectionObserver` for simpler cases.

## Usage

```tsx
import { useElementPositionObserver } from "@fewings/react";

function PositionTracker() {
  const ref = useRef<HTMLDivElement>(null);

  useElementPositionObserver(
    ref,
    (rect) => {
      console.log("Position changed:", rect);
    },
    true
  );

  return <div ref={ref}>Track me!</div>;
}
```

## API

### Parameters

| Name     | Type                                   | Description                                    |
| -------- | -------------------------------------- | ---------------------------------------------- |
| ref      | `React.RefObject<HTMLElement \| null>` | Reference to the element to observe            |
| onChange | `(rect: DOMRect) => void`              | Callback function called when position changes |
| active   | `boolean`                              | Whether the observer should be active          |
