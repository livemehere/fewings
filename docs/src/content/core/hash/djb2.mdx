# djb2

> The `djb2` function is a simple hash function created by Daniel J. Bernstein. It is a non-cryptographic hash function that is used in many applications.

## API

```ts
declare function djb2(str: string): number;
```

## Example

```ts
import { djb2 } from "@fewings/core/hash";

const hash = djb2("Hello, World!");
console.log(hash); // 222957902
```
