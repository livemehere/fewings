# secondToTime

> `secondToTime` is a function that splits seconds into hours, minutes, and seconds.

## API

```ts
/**
 * split seconds into hours, minutes, and seconds
 */
declare function secondToTime(
  second: number,
  to: 'h' | 'm'
): {
  h: number;
  m: number;
  s: number;
};
```

## Example

```ts
import { secondToTime } from '@fewings/core/converter';

console.log(secondToTime(3600, 'h')); // { h: 1, m: 0, s: 0 }
console.log(secondToTime(3660, 'h')); // { h: 1, m: 1, s: 0 }
console.log(secondToTime(3661, 'h')); // { h: 1, m: 1, s: 1 }
console.log(secondToTime(3661, 'm')); // { h: 61, m: 1, s: 1 }
```
