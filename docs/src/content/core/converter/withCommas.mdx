# withCommas

> `withCommas` is a function that takes a number or string and returns a string with commas.

## API

```ts
declare function withCommas(amount: number | string): string;
```

## Example

```ts
import { withCommas } from '@fewings/core/converter';

console.log(withCommas(1000)); // "1,000"
console.log(withCommas(1000000)); // "1,000,000"
console.log(withCommas(1000000000)); // "1,000,000,000"
console.log(withCommas(1000000000000)); // "1,000,000,000,000"
console.log(withCommas(1000000000000000)); // "1,000,000,000,000,000"
console.log(withCommas(1000000000000000000)); // "1,000,000,000,000,000,000"
```

## Limitation

If the number has a decimal part, the function will round the number unexpectedly. (javascript limitation)

```ts
expect(withCommas(123456789.123456789)).not.toBe('123,456,789.123456789');
expect(withCommas(123456789.123456789)).toBe('123,456,789.12345679');
```
