# debounce

> `debounce` is a higher-order function that returns a new function that will only be called after a certain amount of time has passed since the last time it was called.

## API

```ts
declare function debounce<T extends (...args: any[]) => any>(
  fn: T,
  wait: number
): T;
```

## Example

```ts
import { debounce } from '@fewings/core/fp';

const fn = debounce((x: number, y: number) => {
  console.log(x, y);
}, 100);

fn(1, 2);
fn(3, 4);
fn(5, 6);

// Output:
// 5 6
```
