# rand

> The `rand` function generates a random number between a minimum and maximum value.

## API

```ts
declare function rand(min: number, max: number, int?: boolean): number;
```

## Example

```ts
import { rand } from '@fewings/core/math';

console.log(rand(0, 100)); // Random number between 0 and 100
console.log(rand(0, 100, true)); // Random integer between 0 and 100
```
